Change the Number of WooCommerce Products Displayed Per Page

== Description ==

With many WordPress themes, you cannot change the number of WooCommerce products displayed per page.
Now with this plugin you can easily change the number of WooCommerce products displayed per page from your admin control panel.

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Plugin Settings (WooCommerce -> Number of Products Per Page) and fill settings.
